## Выполненная работа
В рамках данной практической работы проведена отладка консольного приложения **«Галактики»** с использованием встроенного отладчика Microsoft Visual Studio.

### Использованные способы отладки Microsoft Visual Studio:
- **Точки останова** (обычные и в цикле `foreach`, а также в конструкторе класса `GType`)
- **Пошаговое выполнение кода**:
  - Step Into (F11) — заход в конструктор `GType` и метод `IterateThroughList`
  - Step Over (F10)
  - Step Out (Shift + F11)
- **DataTips** — всплывающие подсказки при наведении на переменные и свойства объектов
- **Окна отладки**: Locals, Autos, Watch
- **Call Stack** (Стек вызовов)
- **Immediate Window**
- **Hot Reload** и попытка редактирования кода во время отладки
- **Перезапуск отладки** (Ctrl + Shift + F5)
- Изменение значений и исправление кода на лету (Edit and Continue)

### Проделанная работа
1. Запущено приложение и обнаружено несоответствие ожидаемого и фактического вывода.
2. Поставлена точка останова в цикле `foreach` для анализа выводимой строки.
3. С помощью **DataTips** обнаружено, что вместо значения типа галактики выводится имя класса `GType`.
4. Выявлена ошибка типа свойства: `public object GalaxyType { get; set; }` → исправлено на `public GType GalaxyType { get; set; }`.
5. Поставлена точка останова в конструкторе класса `GType`.
6. С помощью пошагового выполнения (F11) и наведения мыши обнаружена **опечатка** в `case 'l'` (строчная буква «L») вместо `case 'I'`.
7. Исправлена опечатка.
8. Добавлены **документирующие XML-комментарии** к классам `Galaxy`, `GType` и методам.
9. После исправлений приложение выводит корректный результат.

**Результат отладки:**

- Tadpole  400,  Spiral
- Pinwheel  25,  Spiral
- Cartwheel, 500,  Lenticular
- Small Magellanic Cloud .2,  Irregular
- Andromeda  3,  Spiral
- Maffei 1,  11,  Elliptical
