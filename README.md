## Выполненная работа

В рамках данной практической работы проведена отладка консольного приложения **«Буквы»** с использованием встроенного отладчика Microsoft Visual Studio.

#### Использованные способы отладки Microsoft Visual Studio:
- **Точки останова** (в цикле `for`)
- **Пошаговое выполнение кода**:
  - Step Into (F11) — заход в метод `SendMessage`
  - Step Over (F10)
  - Step Out (Shift + F11)
  - Run to Click (Выполнить до курсора / Ctrl + F10)
- **Инспекция данных**:
  - **DataTips** (подсказки при наведении мыши на переменные `letters`, `name`, `a[i]`)
  - Окно **Autos** (Автоматические)
  - Окно **Locals** (Локальные)
  - Окно **Watch** (Добавление переменной `name` в наблюдение)
- **Call Stack** (Стек вызовов) — просмотр цепочки вызовов `Main()` → `SendMessage()`
- **Перезапуск отладки** (Ctrl + Shift + F5)
- Просмотр содержимого массива через развёртывание в DataTips / Locals

#### Проделанная работа
1. Создано консольное приложение `GetStartedDebugging`.
2. Вставлен пример кода с массивом букв для формирования имени "fred smith".
3. Поставлена точка останова внутри цикла `for`.
4. Выполнена отладка с полным использованием всех продемонстрированных инструментов.
5. Добавлены **XML-комментарии** к методам `Main` и `SendMessage`.
6. Результаты отладки зафиксированы в Git в ветке.
