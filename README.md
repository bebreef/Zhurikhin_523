## Выполненная работа

В рамках данной практической работы проведена отладка консольного приложения **«Фибоначчи»** с использованием встроенного отладчика Microsoft Visual Studio.

### Использованные способы отладки Microsoft Visual Studio:

- **Точки останова (Breakpoints)** — обычные и условные
- **Пошаговое выполнение кода**:
  - Step Into (F11) — заход в метод
  - Step Over (F10) — выполнение следующей строки
  - Step Out (Shift + F11) — выход из текущего метода
- **Окна отладки**:
  - **Locals** (Локальные переменные)
  - **Autos** (Автоматические)
  - **Watch** (Контрольные значения)
- **DataTips** — всплывающие подсказки с значениями переменных при наведении мыши
- **Immediate Window** (Немедленное окно) — для быстрой оценки выражений
- **Call Stack** (Стек вызовов)
- **Hot Reload** — горячая перезагрузка кода без перезапуска приложения
- Изменение значений переменных прямо во время отладки
