## Описание проекта
Разработано WPF-приложение для расчёта дохода по банковскому вкладу с двумя схемами начисления процентов:
- **Простые проценты**
- **Сложные проценты** 

Приложение содержит валидацию входных данных и подробные XML-комментарии в коде.

---

## Выполненные задачи
- Разработано WPF-приложение (C# + XAML)

- Проведено **ручное тестирование** (TC_01 – TC_05)
