## Описание предметной области

Упрощённая реализация асимметричного шифрования RSA для учебных целей с использованием малых простых чисел.

**Основные требования:**
- Работа с типом `BigInteger` для избежания переполнения
- Шифрование посимвольно (каждый символ как число)
- Простая генерация простых чисел
- Генерация пары ключей (открытый `e`, закрытый `d`, модуль `n`)

**Математическая основа:**
- `n = p × q` (модуль)
- `φ(n) = (p-1) × (q-1)` (функция Эйлера)
- `e` — открытая экспонента (взаимно проста с φ(n))
- `d` — закрытая экспонента (`d × e ≡ 1 (mod φ(n))`)
- Шифрование: `c = m^e mod n`
- Дешифрование: `m = c^d mod n`

## Средства отладки Visual Studio

В процессе разработки и тестирования применялись следующие встроенные средства отладки:

| Средство | Описание применения |
|----------|---------------------|
| **Точки останова (Breakpoints)** | Устанавливались на вход в методы `GenerateKeys`, `Encrypt`, `Decrypt` для контроля параметров |
| **Пошаговое выполнение (F10/F11)** | F10 — построчный проход по обработчикам кнопок; F11 — вход во внутренние методы RsaCipher |
| **Окно «Локальные» (Locals)** | Мониторинг переменных типа `BigInteger` в реальном времени |
| **Условные точки останова** | Остановка только при `!IsPrime(p)` или `new BigInteger(ch) >= n` |
| **Стек вызовов (Call Stack)** | Отслеживание цепочки вызовов при возникновении исключений |
| **Настройки исключений** | Включён режим `Break when thrown` для `ArgumentException` и `FormatException` |
