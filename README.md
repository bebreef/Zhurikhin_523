## Выполненная работа
В рамках данной практической работы проведена отладка консольного приложения **«Галактики»** с использованием встроенного отладчика Microsoft Visual Studio.

### Использованные способы отладки Microsoft Visual Studio:
- **Точки останова (Breakpoints)** — обычные и условные (в том числе внутри цикла `foreach` и конструктора `GType`)
- **Пошаговое выполнение кода**:
  - Step Into (F11) — заход в конструктор `GType` и метод `IterateThroughList`
  - Step Over (F10)
  - Step Out (Shift + F11)
- **Окна отладки**:
  - **Locals** (Локальные переменные)
  - **Autos**
  - **Watch** (Контрольные значения)
- **DataTips** — просмотр значений свойств объектов `Galaxy` и `GType` при наведении
- **Call Stack** (Стек вызовов)
- **Immediate Window** — проверка значений во время паузы
- **Hot Reload** — попытка исправления кода на лету
